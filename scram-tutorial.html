What is Scramjet?

Scramjet is a tool that lets your website act like a tiny “middle-man” for web traffic. People use it for learning how proxies and service-workers work.

How to use it

  1. Download the Scramjet files (from the docs).
  2. Put them in a folder called scram.
  3. Add a file called sw.js to control it.
  4. Publish your site.

This page just explains it. It doesn’t do any hacking — it’s for learning how websites can load service-worker code.
